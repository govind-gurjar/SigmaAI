# SigmaAI

SigmaAI is a Full Stack AI Chat Application built using the MERN Stack and Google Gemini API. It provides secure authentication, AI-powered conversations, chat history, search functionality, and a responsive modern UI.

---

## Live Demo

### Frontend
https://sigma-ai-beige.vercel.app

### Backend
https://sigmaai-backend.onrender.com

---

## Features

- User Authentication (JWT)
-  AI Chat using Google Gemini API
-  Chat History
-  Search Chats
-  Dark & Light Theme
-  Responsive Design
-  Fast React + Vite Frontend
-  MongoDB Database
-  Deployed on Vercel & Render

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/govind-gurjar/SigmaAI.git
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
npm install
npm start
```
